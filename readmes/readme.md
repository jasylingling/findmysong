**🗄 DATENBANK**
<br>
Die Zugangsdaten zur SQL-Datenbank befinden sich unter `model/Db.php`. Gegebenenfalls müssen die Angaben wie DB-Passwort angepasst werden:

```
- "DBSERVER", 'localhost';
- "DBUSER", 'root';
- "DBPASSWORT", ''; ===> xampp: {empty} / mamp: root <===
- "DBNAME", 'findmysong';
```
